# Task 2 — Miglioramento UI/UX Homepage

## Obiettivo
Rendere la homepage più coinvolgente e guidare l'utente verso il suo percorso di apprendimento.

## Azioni

### 1. Hero Section migliorata
- Animazione di benvenuto con il livello attuale dell'utente
- Streak counter prominente con animazione fiamma
- Quick stats: parole imparate, quiz completati, tempo totale
- CTA: "Continua da dove eri rimasto" (riprende ultima attività)

### 2. Daily Challenge
- Sezione "Sfida del Giorno" con un mini-quiz random
- 3 domande veloci, diverso ogni giorno
- XP bonus per completamento

### 3. Progress Dashboard
- Barra progresso per ogni livello (A1-C2)
- Grafico a radar: vocabolario, grammatica, ascolto, lettura, scrittura
- "Punti deboli" suggeriti dall'algoritmo

### 4. Quick Actions Grid
- 6 cards con le attività principali
- Icone animate al hover
- Badge "Nuovo" per contenuti non ancora visti

### 5. Motivational Section
- Frase motivazionale in tedesco con traduzione
- Rotazione giornaliera
- "Sapevi che...?" — fun facts sulla lingua tedesca

### Design
- Mantenere dark theme e colori esistenti
- Animazioni smooth (CSS transitions, no librerie pesanti)
- Mobile-first
